# ⚽ FM26 Mobile Scout

أداة ويب مفتوحة المصدر لاستعراض وتحليل ملفات حفظ لعبة Football Manager 2026 Mobile.
ارفع ملف الحفظ (`.dat`, `.sav`) واحصل على جداول تفاعلية لبيانات اللاعبين، مع تصفية وتصدير وتقارير.

## ✨ الميزات
- رفع مباشر لملف الحفظ (بدون تخزين دائم)
- فك تشفير وتحليل البنية الثنائية (مع وضع تجريبي تلقائي)
- جدول لاعبين مع بحث وتصفية متقدمة
- بطاقة مفصلة لكل لاعب
- مقارنة لاعبين جنباً إلى جنب
- رسوم بيانية تفاعلية (أفضل اللاعبين، توزيع المراكز...)
- تصدير إلى Excel
